# Vitest with MSW library

This project mainly focus on quick utilizing of Vitest with MSW library to mock API request.

## Node

```sh
^20.19.0 || >=22.12.0
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```
